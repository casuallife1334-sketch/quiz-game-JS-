# Режимы игры - Документация

## Обзор

Добавлена поддержка трёх режимов игры:
1. **Своя игра** (Custom Mode) — классический режим со свободным выбором вопросов
2. **Режим обучения** (Training Mode) — вопросы открываются последовательно, при неправильном ответе следующий вопрос остаётся закрытым
3. **Обучалка** (Slideshow Mode) — интерактивный режим с пошаговым слайд-шоу

## Как использовать

### 1. Создание игры в конструкторе

1. Откройте конструктор
2. В верхней панели выберите режим игры:
   - 🎮 **Своя игра** — классический режим
   - 📘 **Обучалка** — режим слайд-шоу с вступлением

### 2. Выбор режима в лобби

После загрузки игры нажмите **"Выбрать режим"** и выберите один из режимов.

## Режим "Обучалка"

### Особенности

Этот режим представляет собой интерактивное слайд-шоу с тремя слайдами:

#### 📸 Слайд 1: Вступление
- Показывается картинка ситуации на весь экран
- Под картинкой заголовок и подробное описание
- Кнопка "Пропустить вступление" для перехода к вопросу
- Ведущий может нажать стрелку вправо для перехода к вопросу

#### ❓ Слайд 2: Вопрос
- Показывается текст вопроса
- Поле для ввода ответа
- Кнопка "Ответить"
- При нажатии "Ответить" происходит проверка:
  - ✅ **Правильно** — запускаются конфетти, автоматический переход к ответу
  - ❌ **Неправильно** — показывается грустный смайлик, автоматический переход к ответу

#### 📝 Слайд 3: Ответ с пояснением
- Показывается правильный ответ
- Текст пояснения
- Картинка пояснения (если есть)
- Прогресс-бар обратного отсчёта
- Автоматическое закрытие через 3-5 секунд

### Конфети

При правильном ответе:
- Запускается анимация конфетти на 200 частиц
- Разноцветные конфетти (16 цветов)
- Анимация длится 5 секунд
- Автоматический переход к следующему вопросу

### Грустный смайлик

При неправильном ответе:
- Показывается анимированный грустный смайлик 😞
- Показывается правильный ответ
- Через 3 секунды возврат к игре

## Логика разблокировки в режиме обучения

### При старте игры
- Разблокируется **первый вопрос первой непустой категории**

### После правильного ответа
- Если есть **следующий вопрос в той же категории** — разблокируется он
- Если это **последний вопрос в категории** — разблокируется первый вопрос следующей категории

### При неправильном ответе
- **Следующий вопрос НЕ разблокируется**
- Игроки могут попробовать ответить на текущий вопрос снова

## Визуальные индикаторы

### Заблокированный вопрос
- 🔒 Иконка замка
- Серый фон
- Низкая прозрачность
- Подсказка при наведении: "Ответьте на предыдущий вопрос, чтобы открыть этот"

### Индикатор в вопросе
Во время просмотра вопроса в режиме обучения отображается подсказка:
> 📘 Режим обучения: при неправильном ответе следующий вопрос остаётся закрытым

## Технические детали

### Новые компоненты

#### `Slideshow.jsx`
Компонент слайд-шоу для режима "Обучалка":
- 3 слайда: вступление, вопрос, результат
- Анимация конфетти при правильном ответе
- Грустный смайлик при неправильном ответе
- Автоматическое переключение слайдов
- Навигационные кнопки

#### `Constructor.jsx` (Обновлён)
- Добавлен переключатель режима игры
- Сохранение `gameMode` в данных игры

### Структура данных

#### gameMode
- `"custom"` — режим "Своя игра"
- `"training"` — режим обучения (вопросы по порядку)
- `"training-slideshow"` — режим обучалка (слайд-шоу)

## Пример использования

```javascript
// Старт игры с режимом обучалки
socket.emit("start-game", {
  roomId: "ABC123",
  game: gameData,
  gameMode: "training"
});

// Открытие вопроса в режиме обучалки
// Автоматически покажет слайд-шоу:
// 1. Вступление с картинкой
// 2. Вопрос с полем ввода
// 3. Ответ с конфетти или грустным смайликом
```

## Тестирование

### Проверка режима обучалки
1. Откройте конструктор
2. Создайте игру с картинками в ситуациях
3. Выберите режим "Обучалка"
4. Сохраните и запустите игру
5. Выберите режим "Обучалка" в лобби
6. Откройте вопрос
7. Проверьте:
   - ✅ Показывается вступление с картинкой
   - ✅ Можно пропустить вступление
   - ✅ Показывается вопрос с полем ввода
   - ✅ При правильном ответе — конфетти
   - ✅ При неправильном ответе — грустный смайлик
   - ✅ Автоматический переход к ответу

### Проверка конфетти
1. Откройте вопрос в режиме обучалки
2. Введите правильный ответ
3. Проверьте:
   - ✅ Конфетти появляются сверху
   - ✅ 200 разноцветных частиц
   - ✅ Анимация длится 5 секунд
   - ✅ Автоматическое закрытие

### Проверка грустного смайлика
1. Откройте вопрос в режиме обучалки
2. Введите неправильный ответ
3. Проверьте:
   - ✅ Показывается грустный смайлик
   - ✅ Показывается правильный ответ
   - ✅ Через 3 секунды возврат к игре

## Известные ограничения

- В режиме обучалки ответ проверяется точным совпадением
- Для лучшего эффекта используйте картинки в ситуациях
- Автоматическое переключение можно отключить в настройках
