# QuizHub - SiGame Style Quiz Game

## 🚀 Быстрый старт

### 1. Запуск сервера и клиента
```bash
npm start
```

Или по отдельности:
```bash
# Терминал 1 - Сервер
npm run server

# Терминал 2 - Клиент
npm run dev
```

## 🎮 Как играть

### Ведущий:
1. Откройте `http://localhost:5173/`
2. Нажмите **"Создать игру"**
3. Введите имя и создайте комнату
4. Скопируйте ссылку на комнату (кнопка "Скопировать ссылку")
5. Отправьте ссылку игрокам
6. Загрузите или создайте игру в конструкторе
7. Нажмите **"Запустить игру"**
8. Выбирайте вопросы на доске

### Игроки:
1. Перейдите по ссылке от ведущего: `http://localhost:5173/?room=XXXX`
2. Если уже играли - подключитесь автоматически
3. Если нет - введите имя и нажмите **"Войти"**
4. Следите за вопросом на экране
5. Отвечайте на вопросы

## 🔧 Синхронизация

Все игроки видят:
- ✅ Одну и ту же игровую доску
- ✅ Открытый вопрос (на весь экран)
- ✅ Счет всех игроков
- ✅ Список игроков в комнате
- ✅ Чат комнаты

## 🛠 Технологии

- **Frontend**: React 18 + React Router + Vite
- **Backend**: Node.js + Express + Socket.IO
- **Стили**: CSS3 с анимациями

## 📝 Примечания

- Только ведущий может выбирать вопросы
- Очки начисляются игроку, который правильно ответил
- Вопрос открывается на весь экран у всех игроков
- При переподключении состояние синхронизируется

## 🐛 Отладка

Откройте консоль браузера (F12) для просмотра логов:
- Подключение к серверу
- Получение состояния игры
- Выбор вопросов
- Обновление счета

Сервер также выводит логи в консоль.
